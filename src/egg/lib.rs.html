19
20## Logging
21
22Many parts of `egg` dump useful logging info using the [`log`](https://docs.rs/log/) crate.
23The easiest way to see this info is to use the [`env_logger`](https://docs.rs/env_logger/)
24crate in your binary or test.
25The simplest way to enable `env_logger` is to put the following line near the top of your `main`:
26`env_logger::init();`.
27Then, set the environment variable `RUST_LOG=egg=info`, or use `warn` or `debug` instead of info
28for less or more logging.
29

89/// A key to identify [`EClass`]es within an

92#[cfg_attr(feature = "serde-1", derive(serde::Serialize, serde::Deserialize))]

96impl From<usize> for Id {
97    fn from(n: usize) -> Id {
98        Id(n as u32)
99    }
100}
101
102impl From<Id> for usize {
103    fn from(id: Id) -> usize {
104        id.0 as usize
105    }
106}
